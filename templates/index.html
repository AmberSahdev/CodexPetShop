two quick ones

How do I generate a pet? add

A pet is two files: a pet.json manifest and a spritesheet.webp with the animation frames laid out side-by-side.

Your pet.json looks like this:

{
  "id": "yourpet",
  "displayName": "Your Pet",
  "description": "A friendly little guy.",
  "spritesheetPath": "spritesheet.webp"
}

Use any tool you like for the spritesheet — Aseprite, Piskel, Procreate. Keep it under 2 MB.

How do I install a pet? add

On any pet's page, hit Download bundle. Drop the unzipped folder into ~/.codex/pets/ and the pet will appear next time Codex starts.
